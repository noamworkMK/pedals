 Pitch Shifter:

System Overview
A frequency-tracking pitch shifter utilizing a CMOS-based architecture. The design avoids digital DSP in favor of a CD4046 Phase-Locked Loop (PLL) and CD4017 decade counter 
for interval sequencing.

Technical Specifications:

Frequency Synthesis: CD4046 PLL tracks input frequency; CD4017 logic handles interval division/multiplication.
Analog Interface: NSL-32 Vactrols integrated for smooth interval transitions.
Calibration: 5x 50k on-board trimmers for precise frequency offset tuning.
Input Stage: TL072 op-amp buffer for high-impedance guitar signal integrity.
Bypass: Relay-based switching system for transparent operation.
